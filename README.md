**Python Network Scanner**
**Overview**

This project is a Python-based network scanner developed for cybersecurity learning purposes.

The scanner uses Python sockets and threading to identify open ports on target systems.

---
**Features**

Port scanning
Multi-threaded scanning
Open port detection
Fast scanning
Command-line interface

---

**Technologies Used**

Python 3	
Socket Library
Threading
Linux CLI

**Example Usage**
python3 scanner.py

Example target:
192.168.56.x

**Example Results**
[OPEN] Port 22
[OPEN] Port 80
